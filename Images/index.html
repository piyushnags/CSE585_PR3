      This is an alphabetical listing of all of the 256x256 images in the CSE/EE 485 image database (~cg485/IMAGES).   When selecting an image, please choose something interesting and meaningful for the project.   For example, when doing edge detection, you should NOT use an image with few gray levels, such as wheel.dat, except as basic proof to yourself that your code works.   You may use an image other than the ones in the database, but make sure it is a 256x256 grayscale image.   In addition, make a raw (.DAT) copy available to me.
